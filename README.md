Btrfs usage monitor
===================

A simple Btrfs usage monitor.


Feature
-------

- Print a warning if Btrfs data usage drops below the free limit percentage.

Usage
-----

    # btrfs-usage-monitor /mnt/btrfs 10
    WARNING /mnt/btrfs free: 752.58GiB (min: 681.47GiB), 9% (limit: 10%)
